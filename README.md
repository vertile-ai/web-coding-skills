## Why Coding Skills

Have you ever faced the scenarios like these when dealing with AI assisted code:

1. spent too much time on planning
2. spent too much time on asking for small modifications from various perspectives
3. spent too much time on understanding AI-generated code

If yes, then `coding-skills` might help you. It is a collection of opininated but battle-tested
instructions at implementation details level for different coding language/framework. Sure, you can
do the spec-driven development, but what if you just want a quick start?

It covers most of common cases of software requirements and provide bottom-up strategy for AI to build modules. Whether or not it's

- Video platform
- 2D games
- 3D games
- Multi-tenant SaaS platform
- Instant Messaging software
- ...
you name it! You should always be able to find one to start with (If not, then we will strive to make it happen for you)

## 2 Major Principles

1. Every skill must be written/verified by human. AI Enhancement is acceptable but no AI slop
2. Every skill is battle tested in real-world environment

## Why NOT Web Coding Skills

This repo is not suitable if you

1. want to pre-design the system

## How does it work

Find the one that suits your need and install the skill using

```bash
npx skills add https://github.com/vertile-ai/web-coding-skills --skills <replace-the-skill-name>
```
